啾啾博客

首先

1.下载安装（node.js) [https://nodejs.org/en] 左边的

2.提前安装 [https://git-scm.com/downloads]

3.自己注册好一个Github账号

具体操作

1.在本地找个目录放博客源文件，注意：路径中不能出现中文

2.在选定的目录中，右键 Git Bash Here

3.使用npm命令安装Hexo，输入：

npm install -g hexo-cli

hexo init blog

cd blog

hexo g

注意：敲完一行按回车

4.cd进入这个目录
