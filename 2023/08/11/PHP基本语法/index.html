PHP语法基本规则

在PHP代码内不可以写标签，PHP的代码可以在标签以内，写完一句加一个分号。

php的输出方式

  • echo 输出字符串，多个字符串用逗号。
  • print() 输出字符串，只能输出一个值。
  • print_r 输出复杂的数据类型，它可以输出复杂的key和值。
1 [1]=>2 [2]=>3) echo [1,2,3] //输出：array ?>
  • var_dump() 可以输出复杂类型，可以输出复杂类型数据的key 和 value, 同时还可以获取value的长度
string(2)'ab' [1] => string(3)'fbb' [2] => string(3)'exo' }

php语法

  • is_dir() - 判断给定文件名是否是一个目录 失败时抛出E_WARNING警告。Note: 此函数的结果会被缓存。

  • ndir() - 返回一个 Directory 类实例

  • nopendir() - 打开目录句柄

  • nchdir() - 改变目录

  • nchmod() - 改变文件模式

  • nchown() - 改变文件的所有者

  • nchgrp() - 改变文件所属的组

  • nfileperms() - 取得文件的权限 失败时抛出E_WARNING警告。Note: 此函数的结果会被缓存。

  • nstat() - 给出文件的信息

  • ngetcwd() - 取得当前工作目录

  • is_link() - 判断给定文件名是否为一个符号连接 失败时抛出E_WARNING警告。Note: 此函数的结果会被缓存。

  • nreadlink() - 返回符号连接指向的目标

  • nsymlink() - 建立符号连接

  • nlink() - 建立一个硬连接

  • nlinkinfo() - 获取一个连接的信息
