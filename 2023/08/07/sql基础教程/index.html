SQL基础教程

什么是SQL

  • SQL指结构化查询语句
  • SQL让我们可以访问数据库
  • SQL是一种标准ANSI计算机语言

建库：create database 库名

建表：create table 表名（定义主键 primary key, 定义自增 auto_increment)

使用某一个数据库：use 库名；

select语句

select语句用于从表中选取数据。
查看。
基本语法：

  • select 列名称 from 表名称
  • selec * from 表名称

distinct

distinct选择自己想看的列
基本语法：

  • select distinct 列名称 from 表名称

where

有条件地从表里选取数据，可将where子句添加到select语句

基本语法：

  • select 列名称 from 表名称 where 列 运算符 值

and和or运算符

AND和OR可在where子语句中把两个或多个条件结合起来。

如果第一个条件和第二个条件都成立，则ADN运算符显示一条记录。

如果第一个条件和第二个条件中只要有一个成立，则OR运算符显示一条记录。

order by

用于根据指定的列对结果集进行排序。

默认按照升序对记录进行排序。

如果需要按照降序对记录进行排序，可以使用desc关键字。

基本语法：

  • insert init 表名称 values （值1，值2，···）
  • insert into table_name (列1，列2，···) values (值1，值2，···)

update

用于修改表中的数据

基本语法：

  • update 表名称 set 列名称 = 新值 where 列名称 = 某值

delete

用于删除表中的行

基本语法：

  • delete from 表名称 where 列名称 = 值

函数

avg()

avg函数返回数值列的平均值，NULL值不包括在计算机中

基本语法：

  • select avg (column_name) from table_name //列名，表名

count()

  1. count(column_name)函数返回指定列的值的数目（null不计入）;

基本语法：

  • select count(column_name) from table_name //列名
  1. count(*)返回表中的记录数；

基本语法：

  • select count(*) from table_name
  1. count(distinct column_name)函数返回指定列的不同值的数目；

基本语法：

  • select count(distinct column_name) from table_name

first()

first()函数返回指定的字段中第一个记录的值

基本语法：

  • select first(column_name) from table_name

last()

last()函数返回指定的字段中最后一个记录的值

基本语法：

  • select last(colum_name) from table_name

max()

max函数返回一列中最大值，NULL值不包括在内。

基本语法：

  • select max(column_name) from table_name

min()

min函数返回一列中的最小值。NULL值不包括在计算机中。

基本语法：

  • select min(column_name) from table_name

group by

group by语句用于结合合计函数，根据一个或多个列对结果集进行分组

基本语法：

  • select column_name,aggregate_function(column_name) from table_name where column_name operator value group by column_name

ucase() // lcase()

ucasc函数把字符的值转换为大写 // 转换成小写

基本语法：

  • select ucase(column_name) from table_name
  • select lcase(column_name) from table_name

len()

len函数返回文本字段中值的长度

基本语法：

  • select len(column_name) from table_name
